Tags Estruturais

Tags O que siginfica
<html>...</html> É a tag de abertura que indica o início de um documento HTML.
<head>...<head> Essa tag HTML é usada para conter informações sobre o documento HTML que não são exibidas diretamente no navegador.
<title>...<title> Define o título da página, que aparece na barra de título do navegador ou na aba.
<body>...<body> É a tag usada para definir o corpo principal de um documento HTML. Todo o conteúdo visível de uma página web é colocado dentro desta tag.
<div>...<div> A palavra "div" é uma abreviação de "divisão". Essencialmente, a tag
é usada para agrupar e organizar outros elementos HTML em seções lógicas ou grupos semânticos.
